# Spacing

Doppel's spacing system is designed to bring consistency and flexibility to every component and layout. Intentional spacing establishes predictable structures, reinforces visual rhythm, and creates a clear hierarchy — ensuring a cohesive and balanced experience across all brand touchpoints.

## Base

Doppel's spacing system uses a base unit of **8px** (0.5rem). This ensures all spacing values are multiples of 8, creating visual harmony and alignment across the design system.

## Usage

Spacing tokens can be applied using Tailwind's utility classes:

- **Padding**: `p-*`, `px-*`, `py-*`, `pt-*`, `pr-*`, `pb-*`, `pl-*`
- **Margin**: `m-*`, `mx-*`, `my-*`, `mt-*`, `mr-*`, `mb-*`, `ml-*`
- **Gap**: `gap-*`, `gap-x-*`, `gap-y-*`

Example:

```jsx
<div className="p-4 gap-4">
  <div className="m-2">Content</div>
</div>
```
