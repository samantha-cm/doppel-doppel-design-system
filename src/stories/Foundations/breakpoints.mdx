# Breakpoints

Doppel's breakpoints establish the structural backbone of responsive design, ensuring the brand experience adapts fluidly and consistently across every screen size.

## Description

Breakpoints define the precise thresholds at which Doppel's layout responds and reorganizes to fit different screen sizes and contexts. These responsive design tokens give designers and developers a shared framework to build interfaces that transition seamlessly — from mobile phones to large desktop displays — preserving visual harmony and usability at every scale.

## Testing Sizes

Doppel's quality assurance (QA) tests are conducted at the following key screen widths to ensure a consistent and polished experience across all breakpoints:

### Mobile

- 320px (small mobile)
- 375px (iPhone SE/12/13)
- 414px (iPhone Plus)

### Tablet

- 768px (iPad Mini)
- 834px (iPad Air/Pro)

### Desktop

- 1024px (small desktop)
- 1280px (standard desktop)
- 1440px (large desktop)
- 1920px (full HD)
