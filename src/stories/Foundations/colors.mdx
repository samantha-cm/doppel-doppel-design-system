# Colors

The Doppel color palette defines the brand’s visual language, ensuring consistent and intentional use of color to strengthen its identity and create a memorable aesthetic. It provides designers and developers with a unified set of tones that maintain visual harmony across all Doppel touchpoints, supporting both accessibility and aesthetic consistency.
